The local AI lab tests which tasks can run privately on shop hardware before becoming part of a workflow.

The main target is useful assistance, not novelty: drafting text from approved specs, summarizing logs, and classifying repeat product data.

## Current tests

- Product description generation
- Supplier note cleanup
- Image tagging experiments
- Short internal summaries
