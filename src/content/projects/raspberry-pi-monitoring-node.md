The monitoring node keeps lightweight telemetry visible without needing a full server for every small signal.

It is intentionally simple: check the network, collect environmental data, and log events that help diagnose workshop issues.

## Signals

- Temperature and humidity
- Network status
- UPS events
- Basic service checks
